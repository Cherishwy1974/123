通解与特解

通解：包含任意常数C的解

$$y = Ce^x$$

（C是任意常数）

特解：满足初始条件的解

$$y(0) = 2 \Rightarrow y = 2e^x$$

  • 通解是无穷多条曲线
  • 特解是其中的一条

阶与次数

阶：最高导数的阶数

  • 一阶：$y' = f(x, y)$
  • 二阶：$y'' = f(x, y, y')$
  • n阶：$y^{(n)} = f(x, y, ..., y^{(n-1)})$

次数：最高阶导数的指数

$$(y'')^2 + y = 0$$

二阶二次方程

总结

微分方程的三要素

  • 定义：含导数的方程
  • 解：通解与特解
  • 分类：阶与次数

微分方程的现实应用

物理应用

  • 自由落体：$m\frac{dv}{dt} = mg$
  • 牛顿冷却律：$\frac{dT}{dt} = -k(T - T_0)$

生物应用

  • 种群增长：$\frac{dP}{dt} = rP$
  • 药物浓度：$\frac{dC}{dt} = -\lambda C$

线性与非线性

线性微分方程

$y' + P(x)y = Q(x)$

关于y的一次方程

非线性微分方程

$y' = y^2 + x$

含y的高次或乘积

  • 线性更易求解
  • 非线性通常需数值方法

显式解与隐式解

显式解

$$y = e^x + C$$

直接表达 $y = f(x)$ 的形式

隐式解

$$x^2 + y^2 = C$$

$F(x, y) = 0$ 的形式

  • 显式解便于计算与应用
  • 隐式解同样有效

药物衰减模型

医学中的微分方程

$$\frac{dC}{dt} = -kC$$

C: 药物浓度，k: 衰减常数

  • 衰减率与当前浓度成正比
  • 解为指数衰减函数
  • 用于计算给药间隔

线性微分方程

线性方程的标准形式

$$\frac{dy}{dx} + P(x)y = Q(x)$$

叠加原理

  • 若y₁、y₂是解
  • 则C₁y₁ + C₂y₂也是解
  • 这是线性方程的重要性质

非线性微分方程

非线性方程的特点

$$\frac{dy}{dx} = y^2$$

（含有y的高次项）

  • 没有通用解法
  • 需要数值方法求解
  • 某些可转化为线性方程

显式解与隐式解

显式解

$$y = f(x)$$

直接表达，便于计算

隐式解

$$F(x, y) = 0$$

间接表达，同样有效

方向场

图解法理解微分方程

  • 在平面上每个点画切线方向
  • 直观展示解曲线的走向
  • 无需求出解析解
  • 帮助理解方程的性质

存在唯一性定理

定理内容

对于 $\frac{dy}{dx} = f(x,y)$

若f和∂f/∂y连续

则通过任意点的解唯一

  • 保证解的存在性
  • 保证解的唯一性
  • 求解的理论基础
