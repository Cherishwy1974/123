一阶线性微分方程

标准形式

$$y' + P(x)y = Q(x)$$

  • 线性：y和y'都是一次的
  • P(x)：y的系数
  • Q(x)：非齐次项

这是工程中最常见的微分方程类型

积分因子法

核心思想

$$\mu(x) = e^{\int P(x)dx}$$

积分因子

解题步骤

  • 求积分因子 $\mu(x)$
  • 两边乘以 $\mu(x)$
  • 左边变为 $(\mu(x)y)'$
  • 积分求解

例题1：求解 $y' - 2y = e^{3x}$

步骤1：找积分因子

$$P(x) = -2$$

$$\mu(x) = e^{-2x}$$

步骤2：两边乘以 $\mu(x)$

$$e^{-2x}y' - 2e^{-2x}y = e^x$$

步骤3：化简并积分

$$(e^{-2x}y)' = e^x$$

$$y = e^{3x} + Ce^{2x}$$

例题2：求解 $y' + y = e^{-x}$

步骤1：找积分因子

$$P(x) = 1$$

$$\mu(x) = e^x$$

步骤2：两边乘以 $e^x$

$$e^x y' + e^x y = 1$$

$$(e^x y)' = 1$$

步骤3：积分求解

$$e^x y = x + C$$

$$y = (x + C)e^{-x}$$

应用：RC电路充电

电路方程

$$RC\frac{dU_C}{dt} + U_C = E$$

$U_C$是电容电压，E是电源电压

化为标准形式

$$\frac{dU_C}{dt} + \frac{1}{RC}U_C = \frac{E}{RC}$$

解

$$U_C = E(1 - e^{-t/RC})$$

电容电压从0逐渐上升到E

总结

一阶线性微分方程

$$y' + P(x)y = Q(x)$$

通解公式

$$y = e^{-\int P(x)dx}\left[\int Q(x)e^{\int P(x)dx}dx + C\right]$$

解题关键

  • 找到积分因子 $\mu(x) = e^{\int P(x)dx}$
  • 两边乘以积分因子
  • 化简为 $(\mu y)' = \mu Q$
  • 积分求解

广泛应用于电路、力学、化学等领域
