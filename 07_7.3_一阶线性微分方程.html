积分因子法

核心思想

$$\mu(x) = e^{\int P(x)dx}$$

积分因子

解题步骤

  • 求积分因子 $\mu(x)$
  • 两边乘以 $\mu(x)$
  • 左边变为 $(\mu(x)y)'$
  • 积分求解

例题1：求解 $y' - 2y = e^{3x}$

步骤1：找积分因子

$$P(x) = -2$$

$$\mu(x) = e^{-2x}$$

步骤2：两边乘以 $\mu(x)$

$$e^{-2x}y' - 2e^{-2x}y = e^x$$

步骤3：化简并积分

$$(e^{-2x}y)' = e^x$$

$$y = e^{3x} + Ce^{2x}$$

例题2：求解 $y' + y = e^{-x}$

步骤1：找积分因子

$$P(x) = 1$$

$$\mu(x) = e^x$$

步骤2：两边乘以 $e^x$

$$e^x y' + e^x y = 1$$

$$(e^x y)' = 1$$

步骤3：积分求解

$$e^x y = x + C$$

$$y = (x + C)e^{-x}$$

应用：RC电路充电

电路方程

$$RC\frac{dU_C}{dt} + U_C = E$$

$U_C$是电容电压，E是电源电压

化为标准形式

$$\frac{dU_C}{dt} + \frac{1}{RC}U_C = \frac{E}{RC}$$

解

$$U_C = E(1 - e^{-t/RC})$$

电容电压从0逐渐上升到E

总结

一阶线性微分方程

$$y' + P(x)y = Q(x)$$

通解公式

$$y = e^{-\int P(x)dx}\left[\int Q(x)e^{\int P(x)dx}dx + C\right]$$

解题关键

  • 找到积分因子 $\mu(x) = e^{\int P(x)dx}$
  • 两边乘以积分因子
  • 化简为 $(\mu y)' = \mu Q$
  • 积分求解

广泛应用于电路、力学、化学等领域

误区与提醒

常见误区：积分因子写错；漏写常数C；未回代检查。

例题1 · 特解

初值条件

$$y(0) = 2$$

求特解

$$y = e^{3x} + Ce^{2x}$$

代入 x=0, y=2

$$2 = 1 + C \Rightarrow C = 1$$

$$y = e^{3x} + e^{2x}$$

例题2 · 题目

求解微分方程

$$\frac{dy}{dx} + y = e^{-x}$$

识别参数

  • P(x) = 1
  • Q(x) = e^(-x)

例题2 · 求解

计算积分因子

$$\mu = e^{\int 1 dx} = e^x$$

方程两边乘以μ

$$e^x \frac{dy}{dx} + e^x y = 1$$

$$(e^x y)' = 1$$

例题2 · 结果

两边积分

$$e^x y = x + C$$

解出y

$$y = (x + C)e^{-x}$$

通解公式

一阶线性方程通解

$$y = e^{-\int P dx} \left( \int Q e^{\int P dx} dx + C \right)$$

  • 先算积分因子 μ = e^(∫Pdx)
  • 再算 ∫Qμ dx
  • 最后除以μ得到y

齐次与非齐次

齐次方程 (Q=0)

$$\frac{dy}{dx} + P(x)y = 0$$

解：$$y = Ce^{-\int P dx}$$

非齐次方程 (Q≠0)

$$\frac{dy}{dx} + P(x)y = Q(x)$$

需要用积分因子法

应用举例

RC电路

$$RC\frac{dU}{dt} + U = E$$

U: 电容电压，E: 电源电压

混合问题

$$\frac{dQ}{dt} + \frac{r}{V}Q = rc_0$$

Q: 溶质量，r: 流速

本节总结

核心知识点

  • 标准形式：dy/dx + Py = Q
  • 积分因子：μ = e^(∫Pdx)
  • 通解公式要记牢
  • 应用：电路、混合问题
