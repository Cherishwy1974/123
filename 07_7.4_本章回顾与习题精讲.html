习题1：可分离变量

题目

$$\frac{dy}{dx} = \frac{x}{y}$$

初始条件：$y(0) = 1$

解答

  • 分离变量：$y dy = x dx$
  • 积分：$\frac{y^2}{2} = \frac{x^2}{2} + C$
  • 通解：$y^2 = x^2 + C_1$
  • 代入初值：$1 = 0 + C_1$，得 $C_1 = 1$
  • 特解：$y = \sqrt{x^2 + 1}$

习题2：一阶线性

题目

$$y' + 3y = 6$$

解答

  • $P(x) = 3$，$Q(x) = 6$
  • 积分因子：$\mu(x) = e^{3x}$
  • 两边乘以 $e^{3x}$：$(e^{3x}y)' = 6e^{3x}$
  • 积分：$e^{3x}y = 2e^{3x} + C$
  • 通解：$y = 2 + Ce^{-3x}$

当 $x \to \infty$ 时，$y \to 2$

习题3：综合应用

题目

$$xy' = y + x^2$$

初始条件：$y(1) = 2$

解答

  • 化为标准形式：$y' - \frac{y}{x} = x$
  • $P(x) = -\frac{1}{x}$，$Q(x) = x$
  • 积分因子：$\mu(x) = e^{-\ln x} = \frac{1}{x}$
  • $(\frac{y}{x})' = 1$
  • 通解：$y = x^2 + Cx$
  • 代入初值：$2 = 1 + C$，得 $C = 1$
  • 特解：$y = x^2 + x$

习题4：冷却问题

题目

物体温度为100℃，放入20℃的环境中，5分钟后温度降到60℃。求温度随时间的变化规律。

牛顿冷却定律

$$\frac{dT}{dt} = -k(T - T_0)$$

$T_0 = 20$℃（环境温度）

解答

  • 分离变量：$\frac{dT}{T-20} = -k dt$
  • 积分：$\ln(T-20) = -kt + C$
  • 通解：$T = 20 + Ce^{-kt}$
  • $t=0$时$T=100$：$C = 80$
  • $t=5$时$T=60$：$k = \frac{1}{5}\ln 2$
  • 特解：$T = 20 + 80e^{-\frac{t\ln 2}{5}}$

总结与提升

解题步骤总结

  • 识别类型：可分离变量还是一阶线性
  • 选择方法：分离变量法或积分因子法
  • 求通解：按步骤计算
  • 代入初值：求特解

常见错误

  • 分离变量时漏掉常数C
  • 积分因子计算错误
  • 忘记代入初始条件

应用领域

电路、力学、热学、化学、经济学等

易错点提示

常见错误：分离变量时未分清项；积分时漏常数；初值代入错误。
