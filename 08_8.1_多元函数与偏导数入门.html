偏导数的概念

偏导数的定义

$$\frac{\partial z}{\partial x} = \lim_{\Delta x \to 0} \frac{f(x+\Delta x, y) - f(x,y)}{\Delta x}$$

几何意义

  • $\frac{\partial z}{\partial x}$：固定y，沿x方向的变化率
  • $\frac{\partial z}{\partial y}$：固定x，沿y方向的变化率

偏导数描述函数在某个方向上的变化快慢

偏导数的计算

计算方法

  • 对x求偏导：把y看作常数
  • 对y求偏导：把x看作常数

例1：$z = x^2 + 3xy + y^2$

$$\frac{\partial z}{\partial x} = 2x + 3y$$

$$\frac{\partial z}{\partial y} = 3x + 2y$$

例2：$z = e^{xy}$

$$\frac{\partial z}{\partial x} = ye^{xy}$$

$$\frac{\partial z}{\partial y} = xe^{xy}$$

高阶偏导数

二阶偏导数

  • $\frac{\partial^2 z}{\partial x^2}$：对x求两次偏导
  • $\frac{\partial^2 z}{\partial y^2}$：对y求两次偏导
  • $\frac{\partial^2 z}{\partial x \partial y}$：先对y再对x
  • $\frac{\partial^2 z}{\partial y \partial x}$：先对x再对y

例：$z = x^3y^2$

$$\frac{\partial^2 z}{\partial x^2} = 6xy^2$$

$$\frac{\partial^2 z}{\partial x \partial y} = 6x^2y$$

偏导数的应用

1. 切平面方程

$$z - z_0 = f_x(x_0,y_0)(x-x_0) + f_y(x_0,y_0)(y-y_0)$$

2. 极值问题

  • 必要条件：$f_x = 0$ 且 $f_y = 0$
  • 驻点：偏导数都为0的点

3. 实际应用

最优化问题、经济学边际分析等

总结

多元函数与偏导数

  • 多元函数：$z = f(x, y)$
  • 偏导数：固定其他变量，对一个变量求导
  • 计算方法：把其他变量看作常数
  • 几何意义：某个方向的切线斜率
  • 应用：切平面、极值、优化问题

偏导数是研究多元函数的基本工具

偏导数定义

对x的偏导

$$\\displaystyle f_x(x_0,y_0)=\\lim_{h \\to 0} \\frac{f(x_0+h,y_0)-f(x_0,y_0)}{h}$$

对y的定义同理

高阶偏导数

二阶偏导数（4种）

$$f_{xx}, \quad f_{yy}$$

$$f_{xy}, \quad f_{yx}$$

重要性质

$$f_{xy} = f_{yx}$$

（连续条件下）

高阶偏导例题

设 $f = x^3 + x^2y + y^3$

第一步：求一阶偏导

$$f_x = 3x^2 + 2xy$$

$$f_y = x^2 + 3y^2$$

高阶偏导计算

第二步：求二阶偏导

$$f_{xx} = 6x + 2y$$

$$f_{xy} = 2x$$

$$f_{yx} = 2x$$

$$f_{yy} = 6y$$

验证：$f_{xy} = f_{yx} = 2x$ ✓

切平面方程

在点 $(x_0, y_0, z_0)$ 处

$$z - z_0 = f_x(x_0,y_0)(x-x_0)$$

$$+ f_y(x_0,y_0)(y-y_0)$$

应用

  • 局部线性近似
  • 工程计算

极值必要条件

驻点（临界点）

$$f_x(x_0, y_0) = 0$$

$$f_y(x_0, y_0) = 0$$

注意

  • 驻点是必要条件
  • 不一定是极值点

极值判别法

判别式

$$D = f_{xx} \cdot f_{yy} - f_{xy}^2$$

判别规则

  • $D > 0, f_{xx} > 0$ → 极小值
  • $D > 0, f_{xx} < 0$ → 极大值
  • $D < 0$ → 鞍点

鞍点

什么是鞍点？

  • 某方向上升
  • 另一方向下降
  • 形状像马鞍

例子

$$f = x^2 - y^2$$

原点是鞍点
