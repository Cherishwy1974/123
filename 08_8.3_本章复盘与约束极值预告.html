第8章知识回顾

核心内容

  • 多元函数：$z = f(x, y)$
  • 偏导数：$\frac{\partial z}{\partial x}$, $\frac{\partial z}{\partial y}$
  • 全微分：$dz = f_x dx + f_y dy$
  • 梯度：$\nabla f = (f_x, f_y)$
  • 方向导数：$\frac{\partial f}{\partial l} = \nabla f \cdot \vec{l}$

多元函数微分学的完整体系

无约束极值

极值的必要条件

$$f_x = 0, \quad f_y = 0$$

驻点（临界点）

极值的充分条件

  • 计算 $A = f_{xx}$, $B = f_{xy}$, $C = f_{yy}$
  • $AC - B^2 > 0$ 且 $A < 0$：极大值
  • $AC - B^2 > 0$ 且 $A > 0$：极小值
  • $AC - B^2 < 0$：鞍点

约束极值问题

什么是约束极值？

在满足约束条件下求极值

求 $f(x,y)$ 的极值

约束条件：$g(x,y) = 0$

实际例子

  • 用固定长度的篱笆围最大面积
  • 用固定材料做最大容积的容器
  • 在预算约束下最大化利润

拉格朗日乘数法

方法步骤

  • 构造拉格朗日函数：

$$L(x,y,\lambda) = f(x,y) + \lambda g(x,y)$$

  • 求偏导数并令其为0：

$$L_x = 0, \quad L_y = 0, \quad L_\lambda = 0$$

  • 解方程组得到候选点
  • 比较函数值确定极值

例题：围篱笆问题

题目

用长度为100m的篱笆围成矩形，求最大面积。

解答

  • 设长为x，宽为y
  • 目标：$f(x,y) = xy$ (最大化)
  • 约束：$g(x,y) = 2x + 2y - 100 = 0$
  • $L = xy + \lambda(2x+2y-100)$
  • $L_x = y + 2\lambda = 0$
  • $L_y = x + 2\lambda = 0$
  • 解得：$x = y = 25$
  • 最大面积：$625 m^2$
