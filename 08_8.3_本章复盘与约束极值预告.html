无约束极值

极值的必要条件

$$f_x = 0, \quad f_y = 0$$

驻点（临界点）

极值的充分条件

  • 计算 $A = f_{xx}$, $B = f_{xy}$, $C = f_{yy}$
  • $AC - B^2 > 0$ 且 $A < 0$：极大值
  • $AC - B^2 > 0$ 且 $A > 0$：极小值
  • $AC - B^2 < 0$：鞍点

约束极值问题

什么是约束极值？

在满足约束条件下求极值

求 $f(x,y)$ 的极值

约束条件：$g(x,y) = 0$

实际例子

  • 用固定长度的篱笆围最大面积
  • 用固定材料做最大容积的容器
  • 在预算约束下最大化利润

拉格朗日乘数法

方法步骤

  • 构造拉格朗日函数：

$$L(x,y,\lambda) = f(x,y) + \lambda g(x,y)$$

  • 求偏导数并令其为0：

$$L_x = 0, \quad L_y = 0, \quad L_\lambda = 0$$

  • 解方程组得到候选点
  • 比较函数值确定极值

例题：围篱笆问题

题目

用长度为100m的篱笆围成矩形，求最大面积。

解答

  • 设长为x，宽为y
  • 目标：$f(x,y) = xy$ (最大化)
  • 约束：$g(x,y) = 2x + 2y - 100 = 0$
  • $L = xy + \lambda(2x+2y-100)$
  • $L_x = y + 2\lambda = 0$
  • $L_y = x + 2\lambda = 0$
  • 解得：$x = y = 25$
  • 最大面积：$625 m^2$

方向导数回顾

定义

$$\frac{\partial f}{\partial \vec{e}} = \nabla f \cdot \vec{e}$$

性质

  • 沿梯度方向最大
  • 垂直梯度方向为零
  • 沿负梯度方向最小

无约束极值回顾

必要条件（驻点）

$$f_x = 0, \quad f_y = 0$$

充分条件（判别式）

$$D = f_{xx} \cdot f_{yy} - f_{xy}^2$$

  • $D > 0, f_{xx} > 0$ → 极小值
  • $D > 0, f_{xx} < 0$ → 极大值
  • $D < 0$ → 鞍点

极值判别法

三步法

  • 第一步：找驻点（$f_x=0, f_y=0$）
  • 第二步：算二阶偏导
  • 第三步：用判别式判断类型

判别式

$$D = f_{xx} f_{yy} - f_{xy}^2$$

约束极值问题

什么是约束极值？

  • 目标函数：$f(x,y)$
  • 约束条件：$g(x,y) = 0$
  • 在约束下优化目标

例子

  • 周长一定，求最大面积
  • 成本约束下最大化产量

拉格朗日乘子法

构造拉格朗日函数

$$L(x,y,\lambda) = f(x,y) - \lambda g(x,y)$$

$\lambda$ 是拉格朗日乘子

求解方程组

$$L_x = 0, \quad L_y = 0, \quad L_\lambda = 0$$

求解步骤

四步法

  • 第一步：构造 $L = f - \lambda g$
  • 第二步：求偏导 $L_x, L_y, L_\lambda$
  • 第三步：令偏导都等于零
  • 第四步：解方程组，判别极值

例题：矩形最大面积

问题

  • 周长为 $P$，求最大面积
  • 设长 $x$，宽 $y$

建立模型

目标：$f = xy$

约束：$g = 2x + 2y - P = 0$

例题求解

构造拉格朗日函数

$$L = xy - \lambda(2x+2y-P)$$

求偏导并令其为零

$$L_x = y - 2\lambda = 0$$

$$L_y = x - 2\lambda = 0$$

$$L_\lambda = -(2x+2y-P) = 0$$

例题结果

解方程组

从前两式：$x = y$

代入第三式：$4x = P$

$$x = y = \frac{P}{4}$$

结论

  • 最优解是正方形
  • 最大面积 = $\frac{P^2}{16}$
