级数的概念

无穷级数

$$\sum_{n=1}^{\infty} u_n = u_1 + u_2 + u_3 + \cdots$$

部分和

$$S_n = \sum_{k=1}^{n} u_k$$

收敛与发散

  • 若 $\lim_{n \to \infty} S_n$ 存在，级数收敛
  • 否则，级数发散

几何级数

定义

$$\sum_{n=0}^{\infty} ar^n = a + ar + ar^2 + \cdots$$

收敛性

  • $|r| < 1$：收敛，和为 $\frac{a}{1-r}$
  • $|r| \geq 1$：发散

级数的性质

基本性质

  • 收敛级数可以逐项相加减
  • 收敛级数乘常数仍收敛
  • 改变有限项不影响敛散性
  • 收敛的必要条件：$\lim_{n \to \infty} u_n = 0$

正项级数判别法

比值判别法

$$\lim_{n \to \infty} \frac{u_{n+1}}{u_n} = \rho$$

  • $\rho < 1$：收敛
  • $\rho > 1$：发散
  • $\rho = 1$：不确定

交错级数

莱布尼茨判别法

若交错级数 $\sum_{n=1}^{\infty} (-1)^{n-1}u_n$ 满足：

  • $u_n$ 单调递减
  • $\lim_{n \to \infty} u_n = 0$

则级数收敛

总结

核心内容

  • 级数的概念：部分和的极限
  • 几何级数：$|r| < 1$ 时收敛
  • 判别法：比值判别法、莱布尼茨判别法

比较判别法

若0 ≤ uₙ ≤ vₙ

∑vₙ收敛 → ∑uₙ收敛

∑uₙ发散 → ∑vₙ发散

比值判别法（达朗贝尔）

$$\rho = \lim_{n\to\infty}\frac{u_{n+1}}{u_n}$$

ρ < 1 → 收敛

ρ > 1 → 发散

ρ = 1 → 不确定

根值判别法（柯西）

$$\rho = \lim_{n\to\infty}\sqrt[n]{u_n}$$

ρ < 1 → 收敛

ρ > 1 → 发散

ρ = 1 → 不确定

交错级数

形式：∑(-1)ⁿuₙ

莱布尼茨判别法

1. uₙ单调递减

2. lim uₙ = 0

→ 级数收敛

绝对收敛与条件收敛

绝对收敛

∑|uₙ|收敛 → ∑uₙ绝对收敛

条件收敛

∑uₙ收敛但∑|uₙ|发散

性质

绝对收敛 → 收敛

例题：判断级数敛散性

$$\sum_{n=1}^{\infty}\frac{1}{n^2}$$

方法

用比较判别法或p级数

p=2>1 → 收敛

例题：比值判别法应用

$$\sum_{n=1}^{\infty}\frac{n!}{n^n}$$

计算

$$\rho = \lim\frac{(n+1)!/(n+1)^{n+1}}{n!/n^n}$$

$$= \lim\frac{1}{(1+1/n)^n} = \frac{1}{e} < 1$$

→ 收敛

级数的应用

• 函数展开：泰勒级数

• 数值计算：近似求和

• 微分方程求解

• 信号处理：傅里叶级数
