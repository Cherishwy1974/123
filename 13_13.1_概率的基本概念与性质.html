随机试验与样本空间

随机试验

结果不确定，但所有可能结果已知的试验

样本空间 Ω

所有可能结果的集合

例：抛硬币

$$\Omega = \{正面, 反面\}$$

例：掷骰子

$$\Omega = \{1, 2, 3, 4, 5, 6\}$$

随机事件

事件

样本空间的子集

必然事件

$\Omega$：一定发生

不可能事件

$\emptyset$：一定不发生

例：掷骰子

  • 事件A：出现偶数 = {2, 4, 6}
  • 事件B：出现大于4 = {5, 6}

概率的定义

古典概型

$$P(A) = \frac{A包含的基本事件数}{样本空间的基本事件总数}$$

例：掷骰子出现偶数

$$P(A) = \frac{3}{6} = \frac{1}{2}$$

概率的性质

  • $0 \leq P(A) \leq 1$
  • $P(\Omega) = 1$
  • $P(\emptyset) = 0$

事件的运算

并事件 A∪B

A或B发生

交事件 A∩B

A和B同时发生

对立事件 $\overline{A}$

A不发生

互斥事件

$A \cap B = \emptyset$：不能同时发生

概率的加法公式

互斥事件

$$P(A \cup B) = P(A) + P(B)$$

一般情况

$$P(A \cup B) = P(A) + P(B) - P(A \cap B)$$

对立事件

$$P(\overline{A}) = 1 - P(A)$$

总结

核心内容

  • 样本空间与随机事件
  • 概率的定义：$P(A) = \frac{m}{n}$
  • 事件的运算：并、交、对立
  • 加法公式：$P(A \cup B) = P(A) + P(B) - P(A \cap B)$
