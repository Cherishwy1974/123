随机试验与样本空间

随机试验

结果不确定，但所有可能结果已知的试验

样本空间 Ω

所有可能结果的集合

例：抛硬币

$$\Omega = \{正面, 反面\}$$

例：掷骰子

$$\Omega = \{1, 2, 3, 4, 5, 6\}$$

随机事件

事件

样本空间的子集

必然事件

$\Omega$：一定发生

不可能事件

$\emptyset$：一定不发生

例：掷骰子

  • 事件A：出现偶数 = {2, 4, 6}
  • 事件B：出现大于4 = {5, 6}

概率的定义

古典概型

$$P(A) = \frac{A包含的基本事件数}{样本空间的基本事件总数}$$

例：掷骰子出现偶数

$$P(A) = \frac{3}{6} = \frac{1}{2}$$

概率的性质

  • $0 \leq P(A) \leq 1$
  • $P(\Omega) = 1$
  • $P(\emptyset) = 0$

事件的运算

并事件 A∪B

A或B发生

交事件 A∩B

A和B同时发生

对立事件 $\overline{A}$

A不发生

互斥事件

$A \cap B = \emptyset$：不能同时发生

概率的加法公式

互斥事件

$$P(A \cup B) = P(A) + P(B)$$

一般情况

$$P(A \cup B) = P(A) + P(B) - P(A \cap B)$$

对立事件

$$P(\overline{A}) = 1 - P(A)$$

条件概率

定义

$$P(A|B) = \frac{P(AB)}{P(B)}$$

B发生的条件下A发生的概率

例：不放回抽球

袋中5红3白，第一次抽到红球后，第二次抽到红球的概率为 4/7

乘法公式

公式

$$P(AB) = P(A)P(B|A)$$

$$= P(B)P(A|B)$$

应用

计算两个事件同时发生的概率

全概率公式

公式

$$P(A) = \sum_{i=1}^n P(B_i)P(A|B_i)$$

B₁,B₂,...,Bₙ是完备事件组

意义

将复杂事件分解为简单情况

贝叶斯公式

公式

$$P(B_i|A) = \frac{P(B_i)P(A|B_i)}{\sum_j P(B_j)P(A|B_j)}$$

意义

由果推因：已知结果推断原因

例题：古典概型

题目

掷两个骰子，点数和为7的概率

解

基本事件总数：36

有利事件：6个

$$P = \frac{6}{36} = \frac{1}{6}$$

例题：条件概率

题目

袋中5红3白，不放回取2球

第一次红球，第二次也红球的概率

解

$$P(A_2|A_1) = \frac{4}{7}$$
