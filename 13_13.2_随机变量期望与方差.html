随机变量的概念

定义

随机变量是样本空间到实数的映射

离散型随机变量

取值有限或可数无穷

例：掷骰子

$$X = \{1, 2, 3, 4, 5, 6\}$$

分布律

$$P(X = k) = p_k, \quad k = 1,2,\ldots,6$$

分布律

定义

$P(X = x_k) = p_k$，描述随机变量的概率分布

性质

  • $p_k \geq 0$
  • $\sum_{k} p_k = 1$

例：掷骰子

$$P(X = k) = \frac{1}{6}, \quad k = 1,2,3,4,5,6$$

数学期望

定义

$$E(X) = \sum_{k} x_k p_k$$

期望是随机变量的平均值

性质

  • $E(C) = C$（常数）
  • $E(aX + b) = aE(X) + b$
  • $E(X + Y) = E(X) + E(Y)$

例：掷骰子

$$E(X) = \frac{1+2+3+4+5+6}{6} = 3.5$$

方差

定义

$$D(X) = E[(X - E(X))^2] = E(X^2) - [E(X)]^2$$

方差衡量随机变量的离散程度

标准差

$$\sigma(X) = \sqrt{D(X)}$$

性质

  • $D(C) = 0$（常数）
  • $D(aX + b) = a^2D(X)$

常见分布的期望和方差

0-1分布

$$E(X) = p, \quad D(X) = p(1-p)$$

二项分布 B(n,p)

$$E(X) = np, \quad D(X) = np(1-p)$$

泊松分布 P(λ)

$$E(X) = \lambda, \quad D(X) = \lambda$$

总结

核心内容

  • 随机变量：样本空间到实数的映射
  • 分布律：$P(X = x_k) = p_k$
  • 期望：$E(X) = \sum x_k p_k$
  • 方差：$D(X) = E(X^2) - [E(X)]^2$
