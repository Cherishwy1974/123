随机变量期望与方差

第13章 视频 13.2

✓ 随机变量的概念

✓ 分布函数

✓ 数学期望

✓ 方差

分布律

定义

$P(X = x_k) = p_k$，描述随机变量的概率分布

性质

  • $p_k \geq 0$
  • $\sum_{k} p_k = 1$

例：掷骰子

$$P(X = k) = \frac{1}{6}, \quad k = 1,2,3,4,5,6$$

数学期望

定义

$$E(X) = \sum_{k} x_k p_k$$

期望是随机变量的平均值

性质

  • $E(C) = C$（常数）
  • $E(aX + b) = aE(X) + b$
  • $E(X + Y) = E(X) + E(Y)$

例：掷骰子

$$E(X) = \frac{1+2+3+4+5+6}{6} = 3.5$$

方差

定义

$$D(X) = E[(X - E(X))^2] = E(X^2) - [E(X)]^2$$

方差衡量随机变量的离散程度

标准差

$$\sigma(X) = \sqrt{D(X)}$$

性质

  • $D(C) = 0$（常数）
  • $D(aX + b) = a^2D(X)$

常见分布的期望和方差

0-1分布

$$E(X) = p, \quad D(X) = p(1-p)$$

二项分布 B(n,p)

$$E(X) = np, \quad D(X) = np(1-p)$$

泊松分布 P(λ)

$$E(X) = \lambda, \quad D(X) = \lambda$$

总结

核心内容

  • 随机变量：样本空间到实数的映射
  • 分布律：$P(X = x_k) = p_k$
  • 期望：$E(X) = \sum x_k p_k$
  • 方差：$D(X) = E(X^2) - [E(X)]^2$

常见分布的期望和方差

二项分布B(n,p)：E(X)=np, D(X)=np(1-p)

泊松分布P(λ)：E(X)=λ, D(X)=λ

期望的性质

E(aX+b) = aE(X)+b

E(X+Y) = E(X)+E(Y)

X,Y独立：E(XY)=E(X)E(Y)

方差的性质

D(aX+b) = a²D(X)

X,Y独立：D(X+Y)=D(X)+D(Y)

协方差与相关系数

Cov(X,Y) = E[(X-E(X))(Y-E(Y))]

ρ = Cov(X,Y)/(σ_X σ_Y)

例题：期望计算

X~B(10,0.3)，求E(X)

解

E(X) = np = 10×0.3 = 3

例题：方差计算

X~B(10,0.3)，求D(X)

解

D(X) = np(1-p) = 10×0.3×0.7 = 2.1

随机变量在数据分析中的应用

• 描述数据分布

• 风险度量

• 预测建模

• 假设检验

学习要点

1. 理解期望和方差的意义

2. 掌握计算方法

3. 熟记常见分布的参数

4. 会用性质简化计算
