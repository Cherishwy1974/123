反函数

探索函数与其反函数的关系和性质

函数选择

$f(x) = 2x + 1$
$f^{-1}(x) = \frac{x-1}{2}$

函数与反函数图像

原函数 f(x)
反函数 f⁻¹(x)
对称轴 y = x

反函数性质

  • 定义域关系 D(f⁻¹) = R(f)
  • 值域关系 R(f⁻¹) = D(f)
  • 对称性 关于 y = x 对称
  • 复合关系 f(f⁻¹(x)) = x
  • 单调性 与原函数相同

数值验证

验证结果:

f(3) = 7

f⁻¹(7) = 3

f(f⁻¹(3)) = 3 ✓

反函数存在条件

• 单射性：函数必须是一一对应的（单射）

• 满射性：在限定的值域内，函数必须是满射的

• 连续性：连续函数在单调区间内必有反函数

• 可逆性：只有双射函数才有反函数

反函数存在的充要条件：$f$ 是双射函数

常见函数的反函数

线性函数

$f(x) = ax + b$
$f^{-1}(x) = \frac{x-b}{a}$
线性函数的反函数仍是线性函数，斜率为原函数斜率的倒数

指数与对数

$f(x) = a^x \leftrightarrow f^{-1}(x) = \log_a x$
指数函数与对数函数互为反函数，是最经典的反函数对

幂函数与根式

$f(x) = x^n \leftrightarrow f^{-1}(x) = \sqrt[n]{x}$
在适当定义域内，幂函数与根式函数互为反函数

三角与反三角

$\sin x \leftrightarrow \arcsin x$
三角函数在限定区间内与反三角函数互为反函数

反比例函数

$f(x) = \frac{1}{x} \leftrightarrow f^{-1}(x) = \frac{1}{x}$
反比例函数的反函数是其自身，具有自反性

三次函数

$f(x) = x^3 \leftrightarrow f^{-1}(x) = \sqrt[3]{x}$
三次函数与立方根函数互为反函数
