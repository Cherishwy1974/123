区间分类

理解数学区间的表示与分类

📏 区间的定义

区间是指在数轴上介于两个实数之间的所有实数的集合。 区间可以用来表示函数的定义域、值域等数学概念。
设 $a, b \in \mathbb{R}$ 且 $a < b$，则区间是满足 $a \leq x \leq b$ 或类似不等式的实数 $x$ 的集合。

🔢 区间的表示方法

闭区间： $[a, b] = \{x | a \leq x \leq b\}$
开区间： $(a, b) = \{x | a < x < b\}$
半开区间： $[a, b) = \{x | a \leq x < b\}$
半开区间： $(a, b] = \{x | a < x \leq b\}$

♾️ 无穷区间

右无穷： $[a, +\infty) = \{x | x \geq a\}$
右无穷： $(a, +\infty) = \{x | x > a\}$
左无穷： $(-\infty, b] = \{x | x \leq b\}$
左无穷： $(-\infty, b) = \{x | x < b\}$
全体实数： $(-\infty, +\infty) = \mathbb{R}$

♾️ 无界区间

右无穷区间 [a, +∞)
从 a 开始向右延伸到正无穷
$[3, +\infty) = \{x | x \geq 3\}$
右无穷区间 (a, +∞)
从 a 的右侧开始向右延伸到正无穷
$(3, +\infty) = \{x | x > 3\}$
左无穷区间 (-∞, b]
从负无穷延伸到 b
$(-\infty, 5] = \{x | x \leq 5\}$
左无穷区间 (-∞, b)
从负无穷延伸到 b 的左侧
$(-\infty, 5) = \{x | x < 5\}$

🎯 特殊区间

全体实数 ℝ
所有实数的集合
$(-\infty, +\infty) = \mathbb{R}$
空集 ∅
不包含任何元素的集合
当 $a > b$ 时，$(a, b) = \emptyset$
单点集 {a}
只包含一个点的集合
$[a, a] = \{a\}$
