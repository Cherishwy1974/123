函数的极值

极值的必要条件：$f'(x_0) = 0$ 或 $f'(x_0)$ 不存在

简单来说：函数在导数为0的点可能取得最大值或最小值

极值计算器

极值分析

一阶导数测试

若 $f'(x_0) = 0$ 且在 $x_0$ 附近：
$f'(x)$ 从正变负 → 极大值
$f'(x)$ 从负变正 → 极小值
通过观察导数符号的变化判断极值类型

二阶导数测试

若 $f'(x_0) = 0$：
$f''(x_0) > 0$ → 极小值
$f''(x_0) < 0$ → 极大值
$f''(x_0) = 0$ → 无法判断
使用二阶导数快速判断极值类型

闭区间上连续函数必定存在绝对最大值和最小值

最优化问题

在约束条件下寻找函数的最大值或最小值

生活中的优化问题：成本最小化、利润最大化、材料用量最少等

最大体积问题

从边长为 a 的正方形纸片四角切去边长为 x 的小正方形，折成无盖纸盒，求最大体积。

围栏问题

最大围栏面积
用固定长度的篱笆围成矩形区域，一边靠墙不需要篱笆，求最大面积。

圆柱体优化

最小表面积圆柱体
给定体积的圆柱体，求表面积最小时的高和底面半径的关系。

最短距离问题

点到曲线的最短距离
求平面上一点到给定曲线的最短距离，以及距离最短时的切点。

运动学中的导数应用

位置 $s(t)$ → 速度 $v(t) = s'(t)$ → 加速度 $a(t) = v'(t) = s''(t)$

简单理解：位置对时间求导得到速度，速度对时间求导得到加速度

运动分析实例

抛物运动

$s(t) = -5t^2 + 20t + 10$
速度：$v(t) = -10t + 20$
加速度：$a(t) = -10$
最高点时间：$t = 2$ 秒
最大高度：$s(2) = 30$ 米

简谐运动

$s(t) = 3\sin(2t) + 1$
速度：$v(t) = 6\cos(2t)$
加速度：$a(t) = -12\sin(2t)$
振幅：3，周期：$\pi$
平衡位置：$s = 1$

指数衰减

$s(t) = 10e^{-0.5t}$
速度：$v(t) = -5e^{-0.5t}$
加速度：$a(t) = 2.5e^{-0.5t}$
速度始终为负（向负方向运动）
加速度为正（减速运动）

经济学中的导数应用

边际成本 = $C'(x)$，边际收益 = $R'(x)$，边际利润 = $P'(x)$

简单理解：边际量就是增加一个单位时的变化量，用导数计算

经济函数分析

成本函数分析

$C(x) = 100 + 50x + 0.5x^2$
分析总成本、平均成本和边际成本的关系，找到最优生产量。

收益函数分析

$R(x) = 200x - 2x^2$
分析收益函数，找到收益最大化的销售量和相应的价格策略。

利润最大化

$P(x) = R(x) - C(x)$
通过分析利润函数的导数，确定利润最大化的生产水平。

需求弹性

$E = -\frac{p}{q} \cdot \frac{dq}{dp}$
使用导数计算需求价格弹性，分析价格变化对需求量的影响。
