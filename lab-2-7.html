不同类型的连续性

连续函数

$f(x) = x^2$
多项式函数在整个定义域内连续，图像没有断点、跳跃或渐近线。

可去间断

$f(x) = \frac{x^2-1}{x-1}, x \neq 1$
在 x = 1 处有可去间断点，可以通过重新定义函数值使其连续。

跳跃间断

$f(x) = \begin{cases} x+1, & x < 0 \\ x-1, & x \geq 0 \end{cases}$
在 x = 0 处有跳跃间断，左右极限存在但不相等。

无穷间断

$f(x) = \frac{1}{x}$
在 x = 0 处有无穷间断，函数值趋向于无穷大。

振荡间断

$f(x) = \sin\left(\frac{1}{x}\right)$
在 x = 0 附近振荡，极限不存在，是振荡间断。

绝对值函数

$f(x) = |x|$
在 x = 0 处连续但不可导，展示连续性与可导性的区别。

函数连续性的定义和性质

连续性的严格定义

定义：函数 $f(x)$ 在点 $x_0$ 处连续，当且仅当满足以下三个条件：

1. 函数值存在：$f(x_0)$ 有定义

2. 极限存在：$\lim_{x \to x_0} f(x)$ 存在

3. 极限等于函数值：$\lim_{x \to x_0} f(x) = f(x_0)$

直观理解：函数在 $x_0$ 处连续意味着当 $x$ 无限接近 $x_0$ 时，$f(x)$ 也无限接近 $f(x_0)$，图像在该点没有"断裂"。

$\varepsilon-\delta$ 定义

函数 $f(x)$ 在点 $x_0$ 处连续，当且仅当：

对任意 $\varepsilon > 0$，存在 $\delta > 0$，使得当 $|x - x_0| < \delta$ 时，有 $|f(x) - f(x_0)| < \varepsilon$

几何意义：对于函数图像上任意一点 $(x_0, f(x_0))$，无论我们要求多小的误差 $\varepsilon$，总能找到一个足够小的邻域 $\delta$，使得在这个邻域内的所有点都落在以 $f(x_0)$ 为中心、半径为 $\varepsilon$ 的"误差带"内。

连续函数的性质

连续函数的四则运算仍然连续
如果函数 f(x) 和 g(x) 在点 x₀ 处连续，那么 f(x)±g(x)、f(x)·g(x) 和 f(x)/g(x)（g(x₀)≠0）也在 x₀ 处连续。

复合函数的连续性

连续函数的复合仍然连续
如果 g(x) 在 x₀ 处连续，f(u) 在 u₀ = g(x₀) 处连续，那么复合函数 f(g(x)) 在 x₀ 处连续。

间断点的分类

可去间断、跳跃间断、无穷间断
第一类间断点：左右极限都存在（可去间断、跳跃间断）；第二类间断点：至少一个单侧极限不存在。

介值定理

连续函数具有介值性
如果函数 f(x) 在闭区间 [a,b] 上连续，且 f(a) ≠ f(b)，那么对于 f(a) 和 f(b) 之间的任意值 k，都存在 c ∈ (a,b) 使得 f(c) = k。
