💡 核心概念

不定积分的本质
不定积分是求导的逆运算，求解所有导数为f(x)的函数族
积分常数 C
积分常数表示原函数有无穷多个，任意常数C的选取不影响导数结果
验证方法
对积分结果求导，如果得到原函数，则积分正确

基本积分公式

$\int x^n dx = \frac{x^{n+1}}{n+1} + C$ (n≠-1)
幂函数积分的基本公式

指数函数积分

$\int e^x dx = e^x + C$
自然指数函数的积分

三角函数积分

$\int \sin x dx = -\cos x + C$
正弦函数的积分

对数函数积分

$\int \frac{1}{x} dx = \ln|x| + C$
倒数函数的积分

换元积分法

$\int f(g(x))g'(x)dx = \int f(u)du$
通过变量替换简化积分计算

分部积分法

$\int udv = uv - \int vdu$
适用于两个函数乘积的积分

部分分式法

$\frac{P(x)}{Q(x)} = \sum \frac{A_i}{(x-a_i)^{n_i}}$
将有理函数分解为简单分式

三角换元法

$x = a\sin\theta, x = a\tan\theta, x = a\sec\theta$
利用三角恒等式简化根式积分

🎯 不定积分的定义

$\int f(x)dx = F(x) + C$
不定积分是求导的逆运算，表示所有导数为f(x)的函数族

🔍 几何意义

面积 = $\int_a^b f(x)dx$
定积分表示曲线与x轴围成的面积，不定积分是面积函数的原函数

⚡ 基本性质

$\int [f(x) + g(x)]dx = \int f(x)dx + \int g(x)dx$
积分具有线性性质，可以逐项积分

🔄 微积分基本定理

$\frac{d}{dx}\int f(x)dx = f(x)$
积分和微分互为逆运算，这是微积分的核心定理

📐 换元积分法

$\int f(g(x))g'(x)dx = \int f(u)du$
通过变量替换简化积分计算，适用于复合函数

🔧 分部积分法

$\int udv = uv - \int vdu$
适用于两个函数乘积的积分，选择合适的u和dv是关键
