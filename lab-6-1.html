什么是向量？

简单理解：向量就是既有大小又有方向的量

向量 = 大小 + 方向

生活例子：

  • 力：推门的力有大小（用多大劲）和方向（往哪推）
  • 速度：汽车行驶有快慢（时速多少）和方向（往哪开）
  • 位移：从A点到B点有距离和方向

向量的表示方法

几何表示

用有向线段表示，箭头表示方向，长度表示大小

$\overrightarrow{AB}$

坐标表示

用坐标形式表示向量的分量

$\vec{a} = (x, y)$ 或 $\vec{a} = (x, y, z)$

向量的基本性质

向量的模长

向量的大小，用 $|\vec{a}|$ 表示

$|\vec{a}| = \sqrt{x^2 + y^2 + z^2}$

单位向量

模长为1的向量

$\vec{e} = \frac{\vec{a}}{|\vec{a}|}$

零向量

模长为0的向量，方向任意

$\vec{0} = (0, 0, 0)$

向量运算

向量加法

对应分量相加

$\vec{a} + \vec{b} = (x_1+x_2, y_1+y_2)$

向量减法

对应分量相减

$\vec{a} - \vec{b} = (x_1-x_2, y_1-y_2)$

数量积

对应分量乘积的和

$\vec{a} \cdot \vec{b} = x_1x_2 + y_1y_2$

例题1：向量表示

题目：已知A(1,2)，B(4,6)，求向量$\overrightarrow{AB}$

$\overrightarrow{AB} = (4-1, 6-2) = (3, 4)$

解释：终点坐标减去起点坐标

例题2：向量模长

题目：求向量$\vec{a} = (3, 4)$的模长

$|\vec{a}| = \sqrt{3^2 + 4^2} = \sqrt{9 + 16} = 5$

解释：这是著名的3-4-5直角三角形

例题3：单位向量

题目：求向量$\vec{a} = (3, 4)$的单位向量

$\vec{e} = \frac{(3, 4)}{5} = (\frac{3}{5}, \frac{4}{5})$

解释：向量除以其模长得到单位向量

例题4：向量加法

题目：计算$\vec{a} = (1, 2)$和$\vec{b} = (3, 4)$的和

$\vec{a} + \vec{b} = (1+3, 2+4) = (4, 6)$

解释：对应分量相加

例题5：数量积

题目：计算$\vec{a} = (2, 3)$和$\vec{b} = (4, 1)$的数量积

$\vec{a} \cdot \vec{b} = 2 \times 4 + 3 \times 1 = 8 + 3 = 11$

解释：对应分量乘积的和
