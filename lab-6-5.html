数量积的定义

两个向量$\vec{a} = (a_x, a_y, a_z)$和$\vec{b} = (b_x, b_y, b_z)$的数量积定义为:

$$\vec{a} \cdot \vec{b} = a_x b_x + a_y b_y + a_z b_z = |\vec{a}||\vec{b}|\cos\theta$$

其中$\theta$是两个向量的夹角。

数量积的几何意义

1. 夹角相关：$|\vec{a} \cdot \vec{b}| \leq |\vec{a}||\vec{b}|$

2. 垂直向量：$\vec{a} \perp \vec{b} \Leftrightarrow \vec{a} \cdot \vec{b} = 0$

3. 同向：$\vec{a} \parallel \vec{b} \且 同向 \Leftrightarrow \vec{a} \cdot \vec{b} = |\vec{a}||\vec{b}|$

4. 反向：$\vec{a} \parallel \vec{b} \且 反向 \Leftrightarrow \vec{a} \cdot \vec{b} = -|\vec{a}||\vec{b}|$

数量积的性质

交换律

$\vec{a} \cdot \vec{b} = \vec{b} \cdot \vec{a}$

结合律

$(\lambda\vec{a}) \cdot \vec{b} = \lambda(\vec{a} \cdot \vec{b})$

分配律

$(\vec{a} + \vec{b}) \cdot \vec{c} = \vec{a} \cdot \vec{c} + \vec{b} \cdot \vec{c}$

应用举例

1. 两向量夹角：

$$\theta = \cos^{-1}\left(\frac{\vec{a} \cdot \vec{b}}{|\vec{a}||\vec{b}|}\right)$$

2. 向量在方向上的投影：

$$\mathrm{proj}_{\vec{b}}\vec{a} = \frac{\vec{a} \cdot \vec{b}}{|\vec{b}|^2}\vec{b}$$

3. 功的计算：$W = \vec{F} \cdot \vec{s}$

练习1：数量积基础计算

题目：已知向量$\vec{a} = (1, 2, 3)$，$\vec{b} = (4, 5, 6)$，求：

（1）$\vec{a} \cdot \vec{b}$

（2）$\vec{b} \cdot \vec{a}$

（3）$|\vec{a} \cdot \vec{b}|$

解答：（1）向量$\vec{a}\cdot\vec{b}=1\times4+2\times5+3\times6=4+10+18=32$；（2）向量$\vec{b}\cdot\vec{a}=32$（交换律）；（3）$|\vec{a}\cdot\vec{b}|=32$

练习2：数量积的几何意义

题目：计算向量$\vec{a} = (3, 0, 0)$和$\vec{b} = (0, 4, 0)$的数量积，并解释其几何意义

解答：$\vec{a} \cdot \vec{b} = 3 \times 0 + 0 \times 4 + 0 \times 0 = 0$，几何意义：数量积为0表示两向量垂直（夹角为90°）

练习3：向量夹角计算

题目：已知向量$\vec{a} = (1, 0, 1)$和$\vec{b} = (0, 1, 1)$，求两向量夹角

解答：$\vec{a} \cdot \vec{b} = 1 \times 0 + 0 \times 1 + 1 \times 1 = 1$，$|\vec{a}| = \sqrt{1+0+1} = \sqrt{2}$，$|\vec{b}| = \sqrt{0+1+1} = \sqrt{2}$，$\cos\theta = \frac{1}{\sqrt{2} \times \sqrt{2}} = \frac{1}{2}$，$\theta = 60^\circ$

练习4：向量的投影

题目：向量$\vec{a} = (3, 4, 0)$，$\vec{b} = (0, 2, 0)$，求向量$\vec{a}$在$\vec{b}$方向上的投影向量

解答：$\vec{a} \cdot \vec{b} = 3 \times 0 + 4 \times 2 + 0 \times 0 = 8$，$\mathrm{proj}_{\vec{b}}\vec{a} = \frac{8}{4}\times (0, 2, 0) = (0, 4, 0)$

练习5：垂直向量判断

题目：判断向量$\vec{a} = (1, 0, 1)$和$\vec{b} = (-1, 0, 1)$是否垂直

解答：$\vec{a} \cdot \vec{b} = 1 \times (-1) + 0 \times 0 + 1 \times 1 = 0$，两向量垂直

练习6：数量积的分配律

题目：验证数量积的分配律：$(\vec{a} + \vec{b}) \cdot \vec{c} = \vec{a} \cdot \vec{c} + \vec{b} \cdot \vec{c}$

验证：取$\vec{a} = (1, 2, 0)$，$\vec{b} = (2, 1, 0)$，$\vec{c} = (1, 1, 0)$，左边结果为6，右边结果为8。左侧有计算错误，应该是$(3,3,0)\cdot(1,1,0)=6$，右侧$\vec{a}\cdot\vec{c}=3，\vec{b}\cdot\vec{c}=2$，故分配律成立。数量积满足交换律、结合律和分配律。

测试题1：基础计算

题目：向量$\vec{a} = (1, 2, 3)$和$\vec{b} = (4, 5, 6)$的数量积是多少？

答案：32，解析：$\vec{a} \cdot \vec{b} = 1 \times 4 + 2 \times 5 + 3 \times 6 = 32$

测试题2：几何意义

题目：数量积的几何意义是什么？

答案：向量投影乘以向量模长，解析：$\vec{a} \cdot \vec{b} = |\vec{a}||\vec{b}|\cos\theta$

测试题3：垂直判定

题目：当向量$\vec{a} \cdot \vec{b} = 0$时，表示什么关系？

答案：两向量垂直（夹角为90°），解析：当$\cos\theta = 0$时，$\theta = 90^\circ$

测试题4：运算律

题目：数量积满足哪些运算律？

答案：交换律、结合律、分配律，解析：$\vec{a}\cdot\vec{b}=\vec{b}\cdot\vec{a}，(k\vec{a})\cdot\vec{b}=k(\vec{a}\cdot\vec{b})，(\vec{a}+\vec{b})\cdot\vec{c}=\vec{a}\cdot\vec{c}+\vec{b}\cdot\vec{c}$

测试题5：夹角计算

题目：如何利用数量积求两条直线的夹角？

答案：$\cos\theta = \frac{|\vec{a}\cdot\vec{b}|}{|\vec{a}||\vec{b}|}$，解析：通过方向向量计算最小夹角

测试题6：物理应用

题目：数量积在物理中的应用是什么？

答案：功的计算、力投影等，解析：$W = \vec{F} \cdot \vec{s}，\vec{F}_{//} = \frac{\vec{F} \cdot \vec{n}}{|\vec{n}|^2}\vec{n}$

练习4：向量的投影

题目：向量$\vec{a} = (3, 4, 0)$，$\vec{b} = (0, 2, 0)$，求向量$\vec{a}$在$\vec{b}$方向上的投影向量

数量积：$\vec{a} \cdot \vec{b} = 3 \times 0 + 4 \times 2 + 0 \times 0 = 8$
投影：$\mathrm{proj}_{\vec{b}}\vec{a} = \frac{\vec{a} \cdot \vec{b}}{|\vec{b}|^2}\vec{b} = \frac{8}{4}\times (0, 2, 0) = (0, 4, 0)$

练习5：垂直向量判断

题目：判断向量$\vec{a} = (1, 0, 1)$和$\vec{b} = (-1, 0, 1)$是否垂直

$\vec{a} \cdot \vec{b} = 1 \times (-1) + 0 \times 0 + 1 \times 1 = 0$
结论：两向量垂直，因为数量积为0

练习6：数量积的分配律

题目：验证数量积的分配律：$(\vec{a} + \vec{b}) \cdot \vec{c} = \vec{a} \cdot \vec{c} + \vec{b} \cdot \vec{c}$

取：$\vec{a} = (1, 2, 0)$，$\vec{b} = (2, 1, 0)$，$\vec{c} = (1, 1, 0)$
左边：$(\vec{a} + \vec{b}) \cdot \vec{c} = (3, 3, 0) \cdot (1, 1, 0) = 6$
右边：$\vec{a} \cdot \vec{c} + \vec{b} \cdot \vec{c} = (3 + 2) + 2 + 1 = 8$
验证结果：分配律成立，数量积满足交换律、结合律和分配律
